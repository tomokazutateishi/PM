# タスク管理システム

## フォルダ構成

- `active/` - 進行中のタスク
- `completed/` - 完了したタスク
- `archive/` - アーカイブされたタスク

## 使い方

### タスクの作成

1. `active/` フォルダに新しいMarkdownファイルを作成
2. ファイル名は `YYYY-MM-DD-タスク名.md` の形式で作成
3. テンプレートを使用してタスク内容を記述

### タスクの進捗管理

各タスクファイルには以下の情報を記録：

- **ステータス**: 未着手 / 進行中 / レビュー中 / 完了
- **優先度**: 高 / 中 / 低
- **期限**: YYYY-MM-DD
- **担当者**: 名前
- **説明**: タスクの詳細
- **進捗**: 0-100%
- **メモ**: 作業中のメモや気づき

### タスクの移動

- タスクが完了したら `completed/` に移動
- 長期間保留になったタスクは `archive/` に移動

## テンプレート

新しいタスクを作成する際は、`template.md` をコピーして使用してください。
