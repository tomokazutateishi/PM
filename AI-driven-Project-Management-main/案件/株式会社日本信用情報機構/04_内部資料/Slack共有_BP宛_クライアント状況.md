# Slack共有：BP宛（日本信用情報機構様のご連絡内容）

営業から共有があったクライアント連絡を、パートナー（ideaさん）向けに転記するための文面です。

---

**Slack用（コピペ）**

```
お世話になっております。

日本信用情報機構様の件で、クライアント様から営業に以下のご連絡があったため、パートナー様にも共有させてください。

・ご提案書をお送りいただいたことへのお礼と、ご連絡が遅くなったことへのお詫びのご連絡をいただいています。
・弊社の状況として、他業務との兼ね合いから**今期中のアクセシビリティ診断の実施は難しい**と判断されたとのことです。
・今期は**情報収集にとどめ**、現状・来期の実施可否検討に向けて社内の企画準備を進めている状況とのことです。
・お送りした資料を拝見いただき社内調整を進めていただく予定で、不明点等あればまたご連絡いただく旨です。

つきましては、本件は今期の実施見込みは立たない前提で進めていただき、来期以降の動きが出てきた際には改めてご相談させていただく形になりそうです。
ご確認のほどよろしくお願いいたします。
```

---

```
お世話になっております。

パートナー様にも共有いただきたく、ご連絡しました。来期以降の動きが出た際は改めてご相談します。よろしくお願いいたします。
```
