# 株式会社日本信用情報機構案件 フォルダ構成

## プロジェクト概要

- **プロジェクト名**: 株式会社日本信用情報機構_アクセシビリティ診断
- **PJステータス**: 提案検討中
- **概要**: アクセシビリティ診断（75ページフル or 10ページ版の2パターン検討中）
- **開始時期**: （未定）

## フォルダ構成

### 01_プロジェクト管理ドキュメント
PMBOKに沿ったプロジェクト管理ドキュメント

- 議事録・プロジェクト憲章・WBS・リスク登録簿など

### 02_提案書
クライアント向け提案書とその関連資料

- 見積書（75ページフル版 / 10ページ版）
- 報告会/研修オプション検討資料

### 04_内部資料
社内でのレビューや打ち合わせ用の資料

## ネクストアクション

### クライアント側
- 調査対象ページ（約75ページ）のリストを整理して提示する
- 「75ページフル」か「代表10ページ」か、社内で方針を検討する

### ARI側
- 「75ページフル」と「10ページ版」の2パターンで調査見積を作成する
- 報告会／研修（更新担当やベンダー向け）のオプションの有無・参考価格帯を検討する
- 提案提出時期は、クライアント希望として「1月中に概算把握」。難しければ概算でも可。最終予算確定は2月下旬
