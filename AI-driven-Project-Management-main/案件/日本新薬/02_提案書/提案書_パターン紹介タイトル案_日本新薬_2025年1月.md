# パターン紹介スライド タイトル案（日本新薬案件）

## スライドの内容確認

3つのパターン（A、B、C）を紹介するスライドのタイトル案です。

### スライドの内容
- コーポレートサイトの改善には、ご予算の規模や関与の深さ、社内での意思決定の進めやすさに応じて、最適なアプローチをご選択いただけるよう、3つのパターンを整理いたしました。
- Aパターン：段階的リニューアル型
- Bパターン：保守・運用型
- Cパターン：保守+スポット対応

---

## タイトル案

### 案1：選択肢を強調するタイプ

**タイトル：**
```
ご提案パターン
```

**サブタイトル（オプション）：**
```
ご予算の規模や関与の深さ、社内での意思決定の進めやすさに応じて、最適なアプローチをご選択いただけます
```

**特徴：**
- シンプルで分かりやすい
- パターン紹介であることが明確
- 選択肢を提示する意図が伝わる

---

### 案2：改善方針を強調するタイプ

**タイトル：**
```
改善の進め方：3つのパターン
```

**サブタイトル（オプション）：**
```
ご予算の規模や関与の深さ、社内での意思決定の進めやすさに応じて、最適なアプローチをご選択いただけます
```

**特徴：**
- 「改善の進め方」という視点を明確化
- 3つのパターンであることが分かる
- 改善コンセプト・方針のセクション内であることが自然

---

### 案3：柔軟性を強調するタイプ

**タイトル：**
```
状況に応じた改善アプローチ
```

**サブタイトル（オプション）：**
```
ご予算の規模や関与の深さ、社内での意思決定の進めやすさに応じて、3つのパターンから最適なアプローチをご選択いただけます
```

**特徴：**
- 柔軟性と選択肢の多様性を強調
- 状況に応じた対応が可能であることを示す
- より戦略的な印象を与える

---

## 推奨

**案1「ご提案パターン」**を推奨します。

**理由：**
- シンプルで分かりやすい
- スライドの内容（3つのパターン紹介）と一致
- 既存のスライド構成との整合性が高い
- 視覚的に分かりやすい（表形式のパターン紹介と相性が良い）
