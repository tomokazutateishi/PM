# 案件管理（`案件/`）

## 共通テンプレ

- `PMBOKRULE/`: 共通テンプレ（`議事録_template.md` / `プロジェクト憲章_template.md` / `リスク登録簿_template.md` など）

## 新規案件の作り方（最小）

- **案件フォルダ作成**: `案件/{案件名}/`
- **最低限置く**（推奨）:
  - `README.md`（フォルダ構成/リンク集）
  - `01_プロジェクト管理ドキュメント/`（憲章/WBS/リスク/議事録）
  - `02_提案書/`（提案書・スライド関連）
  - `04_内部資料/`（社内用の台本/FBログなど）

## タスクとの連携

全案件横断で追うタスクは `../タスク/タスク一覧.md` に集約し、案件側の該当資料へリンクを付ける運用を推奨します。
