## 3. 議題と討議内容

### 議題1：御社の現状と課題

| 討議内容 |
|----------|
| レディクル経由の面談は初めて。制作会社の選定は各所で進行中だが、未だ決定していない。 |
| 現行サイトは「古い・昭和っぽい」と上層部から指摘。大手・地元他社は今風に変わっており、乗り遅れ感がある。社内にデザイナーはおらず、制作会社に依頼している。 |
| 運用面：Web広告・QRの遷移先がすべてHPになっており、マンションの第一印象を損ねている。アクセスはスマホ:PC≒7:3のため、スマホ重視のデザインが必要。 |
| 年間3〜4物件の新築マンションサイトを制作。現行ベンダーへの年間費用は約300万円（デザイン〜構築〜運用込み）。1物件あたり約100万円程度の感覚（御社認識では現実的には低い）。 |

### 議題2：本案のスコープ・デザイン方針

| 討議内容 |
|----------|
| 今回の依頼範囲は**新築マンション物件サイト 1件分のデザイン**（新規制作）。既存サイトは据え置き。 |
| デザインの雰囲気は**リゾート寄り・開放感・明るい**（那覇空港近・県外向け）。都心シックは不要。1パターンでよい。 |
| コーポレートサイトは15ページ程度・構成は現状のまま。更新は制作会社へ依頼（御社ではほとんど触らない）。 |
| 依頼範囲は**デザインのみ**。構築・保守は既存の制作会社に依頼予定。 |

### 議題3：物件・スケジュール

| 討議内容 |
|----------|
| 今月20日販売開始の物件は間に合わない。**次の物件（7月前後販売開始）**を対象とするのが現実的。 |
| 販売開始までにページが仕上がっていればよく、2〜3ヶ月前の先行公開は必須ではない。予告広告は簡易ページで対応可。 |
| デザインは課題整理→ワイヤーフレーム→合意→ビジュアルの流れで、着手から**約3ヶ月**を想定。御社の確認は2日以内、最終意思決定は社内にいるため遅くない想定。 |
| 弊社の着手はリソース次第で**最短3月開始**が目安。3月開始であれば6月頃デザイン完了→7月販売開始に間に合わせるイメージ。 |

### 議題4：予算

| 討議内容 |
|----------|
| コーポレートサイト：デザインのみで100〜300万円の検討可能。 |
| 物件サイト：1件あたり100〜300万円でできればよいという希望。500万円超は厳しい。一方で「高くてもやろう」という雰囲気もあり、**金額よりクオリティ重視**。見積提出後に上層部の反応を確認する。 |

### 議題5：素材・納品

| 討議内容 |
|----------|
| 物件情報・CGパース・写真・間取りは御社から提供可能。間取り以外のルームプラン（階別図等）は弊社でデザインする想定。 |
| 風景素材等は制作会社経由で調整。写真の加工は元画像を大きく崩さない範囲で対応。 |

### 議題6：選定の進め方

| 討議内容 |
|----------|
| まずは**実績（作ったサイト）**を見たい。不動産に限らず、雰囲気が似たサイトのデザインで可。提案デザインが用意できればなおよい。 |
| 選定・決裁は上長・社長含む広報部。決裁はスムーズな方。「この会社でやりたい」と伝えれば「じゃあやろう」となりそうな感覚。 |
| コンペの形式・コンペ費・期限は未定。弊社から実績をまとめて送付し、そのうえで見積・提案を出す流れを想定。**できるだけ早く決めたい**との意向。 |

## 4. 決定事項
