# タスク管理（`タスク/`）

このフォルダは **全案件横断の「いまやること」管理** の入口です。  
案件ごとの成果物（議事録/憲章/WBS/提案書など）は `案件/` 側に保存し、ここには **一覧（俯瞰）** を置きます。

## 運用ルール（最小）

- **追加**: 会議後・レビュー後に `タスク一覧.md` へ追記（案件フォルダへのリンクを添える）
- **更新**: ステータス更新は「毎日/週次」で棚卸し
- **完了**: 完了タスクは `完了` にして行を残す（監査ログとして残す）か、下部の「完了」セクションへ移動
- **粒度**: 1タスク=半日〜2日で終わる粒度が目安（大きいものは分割）

## ステータス定義

- **未着手**: まだ開始していない
- **進行中**: 着手済み
- **保留**: 依存/判断待ち/相手待ち
- **完了**: 完了済み
