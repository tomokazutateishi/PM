## エージェント構成

### 使用しているエージェント構成
**シンプル5エージェント構成**（MULTI_AGENTS_SIMPLE.md）

1. **社長（agent-ceo）**: 戦略的意思決定・最終承認
2. **PM（agent-pm）**: 全体計画・進捗管理
3. **マーケター（agent-marketer）**: 市場調査・ユーザー要件
4. **プログラマー（agent-programmer）**: コード実装
5. **コードチェック（agent-codecheck）**: コードレビュー

---

## プロジェクトの流れ

1. **アイデア出し**（02_アイデア出し/）
   - 社長とマーケターが協調してアプリアイデアを検討
   - 在宅ワーク効率化アプリを第1候補として選定

2. **プロジェクト承認**（03_プロジェクト/01_プロジェクト管理ドキュメント/意思決定/）
   - 社長がプロジェクトを承認（条件付き）

3. **市場調査・ユーザー調査**（03_プロジェクト/02_マーケティング/）
   - マーケターが市場調査とユーザー調査を実施

4. **要件定義**（03_プロジェクト/03_要件定義/）
   - マーケターが機能要件・非機能要件を定義

5. **プロジェクト計画**（03_プロジェクト/01_プロジェクト管理ドキュメント/プロジェクト計画/）
   - PMがプロジェクト計画を作成
